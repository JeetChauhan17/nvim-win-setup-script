## BEFORE EXECUTING :
- run powershell as admin and then run : 
```ps1
Set-ExecutionPolicy Bypass -Scope Process -Force
```

## THEN RUN
```
.\setup.ps1
```
 ### This will install the following : 
 1. Windows Terminal
 2. Git
 3. PNPM
 4. Yarn
 5. python
 6. NodeJS
 7. nvim-config (neovim now included, install seperately)
